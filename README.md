# Homelab Dashboard

Персональный дашборд для homelab-инфраструктуры — быстрый доступ ко всем сервисам с проверкой статуса.
Наверное...

**Возможности:**
- Проверка доступности сервисов с авто-перепроверкой каждые 2.5 минуты
- Фильтрация по категориям и по Yggdrasil-адресам
- Перетаскивание карточек — порядок сохраняется в браузере
- Поиск по всем сервисам `Cmd/Ctrl+K`
- Три раздела: публичные сервисы, локальная сеть, внешние VPS
- Тёмная/светлая/авто тема с анимацией переключения
- PWA — устанавливается как приложение на телефон или десктоп

---

## Конфигурация

Все сервисы описаны в одном файле: `app/composables/useLinks.ts`

---

## Запуск для разработки

```bash
npm install
npm run dev
```

---

## Деплой: Docker

Собирает и раздаёт статику через nginx:alpine в одном контейнере.

```bash
docker compose up -d --build
```

Сайт будет доступен на `http://localhost:4444`

Для production рекомендуется поставить за reverse proxy (например Nginx Proxy Manager) с SSL-сертификатом.

---

## Деплой: статический хостинг

Подходит для любого хостинга статических файлов (GitHub Pages, Netlify, Vercel, свой nginx и т.д.)

```bash
npm install
npm run generate
```

Содержимое папки `.output/public/` загрузить на хостинг.
